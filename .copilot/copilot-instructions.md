# Copilot Instructions

## Scope of Changes

You may **only** modify files and directories that are managed by [yadm](https://yadm.io/) in this environment.

### Managed directories and files

| Path | Type |
|------|------|
| `~/.config/` | directory |
| `~/.local/bin/` | directory |
| `~/.local/share/gnupg/` | directory |
| `~/.local/share/public.asc` | file |
| `~/.local/share/yadm/` | directory |
| `~/.ssh/` | directory |
| `~/.copilot/copilot-instructions.md` | file |
| `~/.profile` | file |
| `~/.zprofile` | file |
| `~/README.md` | file |

> **Do not** create, edit, or delete any files outside of these paths.
> Run `yadm list` to get the current authoritative list of tracked files.
